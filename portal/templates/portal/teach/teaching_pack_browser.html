Welcome to the teaching materials browser! Here you can browse, preview and download all the materials available to help structure your pupils progress through rapid router. For each stage of education the materials are divied into four catagories:Sessions, Resource sheets, Assessments, Assets. Below is a summary of what you can find in these materials.

Sessions

These are lesson plans that will help guide your class through the rapid router levels. Along with details of activities, each plan indicates the Rapid router levels that should be looked at by pupils at each lesson as well as any resource sheets or videos that can be used along side the plan.

Resource Sheets

These sheets contains series of exercises to be used along side the lesson plans. These include completing algorithms on paper as well as drawing out the path the van would take for a given algorithm.

Assessments

This folder contains a sheet for pupils to self assess their knowledge of core concepts and key words aswell as a sheet for teachers to keep record of pupils' understanding.

Assets

These folders contains Code wall cards with useful programming words and phrases. This is also where to find blockly and python cut outs for building algorithms on paper.
